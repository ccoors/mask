<h2> what the plugin actually do?</h2>


<h3>Push Button:</h3>
<h4>"cd "project path" && git add . && git commit -m "the commit you write on "Commit" Entry" && git push -u origin "Branch you choose from Branch_Options" "</h4>


<h3>Pull Button:</h3>
<h4>"cd "project path" && git reset --hard && git pull"</h4>
